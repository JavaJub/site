# Аналитика — приватная, без cookies и ПДН (вариант B)

Цель: понимать **куда заходят и сколько времени проводят** пользователи, **не обрабатывая
персональные данные** (без cookies, без хранения IP) — чтобы не подпадать под 152-ФЗ /
уведомление РКН, пока оно не подано.

По умолчанию аналитики **нет**: переменные `PUBLIC_ANALYTICS_*` пустые → ни одного скрипта,
ни одного стороннего запроса, ноль cookies. Сайт остаётся «чистым».

## Рекомендуемый путь — self-hosted Umami (cookieless)

[Umami](https://umami.is) — open-source, **без cookies**, не хранит IP (хеширует
`IP+UA` в дневной идентификатор сессии и выбрасывает сырой IP), даёт страницы, источники,
длительность, пути, устройства. Дашборд — твой.

1. Подними Umami на своём сервере (**желательно в РФ** — снимает вопрос 242-ФЗ),
   например через Docker (`ghcr.io/umami-software/umami`). Нужен Postgres/MySQL.
2. Заведи сайт в Umami → получишь `website-id` и URL скрипта (`/script.js`).
3. В `.env` сайта пропиши:
   ```
   PUBLIC_ANALYTICS_SRC=https://analytics.твойдомен.ru/script.js
   PUBLIC_ANALYTICS_ID=<website-id>
   ```
4. Пересобери сайт (`npm run build`). Тег подключается с `data-do-not-track="true"`.

> Plausible / GoatCounter (тоже cookieless, self-hosted) подойдут аналогично — у Plausible
> атрибут называется `data-domain`, поправь одну строку в `src/layouts/Base.astro`.

## Вариант без JS — анализ логов (GoAccess)

Если сайт за Caddy/Nginx, можно вообще без клиентского кода:
- включи логи доступа, **анонимизируй IP** (например, обнуляй последний октет),
- гоняй [GoAccess](https://goaccess.io) по логам → отчёт по страницам, источникам, ботам.

Ноль cookies, ноль клиентских запросов, минимум данных. Не даёт точное «время на странице»,
но показывает посещаемость и популярные разделы.

## Чего НЕ делаем сейчас

**Яндекс.Метрика / Google Analytics** — обрабатывают ПДН (IP, cookies, поведение). Подключать
их можно только после подачи уведомления в РКН + публикации Политики конфиденциальности +
баннера согласия. Интеграция готова к этому: добавим счётчик за тем же флагом, когда будешь готов.
